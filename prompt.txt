=== GABUTPOC — APK DISTRIBUTION ===

Server SRS + relay + dashboard SUDAH berjalan di laptop lain.
Hanya distribusi APK ke device target.

== BUILD ULANG (jika IP server berbeda) ==
- Edit android/gradle.properties, set SRS_HOST=10.103.105.79:1985
- cd android && gradlew.bat assembleDebug
- APK di: app\build\outputs\apk\debug\
- Pilih: app-universal-debug.apk (untuk semua device)

== INSTALL ==
- Via adb: adb install app-universal-debug.apk
- Manual: kirim APK ke device, buka file manager, tap install
- Jika Play Protect blokir: Settings → Play Protect → matikan scan

== STREAMING ==
- Buka app → tap "Aktifkan Stream"
- Izinkan screen recording + notifikasi
- Status: kuning (connecting) → hijau (live)

== LIHAT ==
- Browser ke https://10.103.105.79/dashboard.html
- Klik device → stream muncul

== REMOTE CONTROL ==
- Tap "Aktifkan Full Control" → buka Accessibility → enable GabutPoC
- Dashboard → Remote → kirim perintah (tap/swipe/key)

== PORT ==
- 1985 TCP: WHIP/WHEP/API
- 8000 UDP: WebRTC media
- 8088 TCP: Dashboard
- 8092 TCP: Relay WebSocket

== TROUBLESHOOT ==
- Streaming stuck kuning: cek firewall server (UDP 8000)
- Video hitam: CANDIDATE IP server salah
- Remote tidak bisa: Accessibility Service belum aktif
